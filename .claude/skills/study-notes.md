# Hello-Agents 学习笔记助手

当用户在学习 Hello-Agents 项目时，帮助用户记录和整理学习笔记。

## 触发条件

当用户：
- 询问项目中的知识点
- 请求解释某个概念
- 明确要求记录笔记

## 执行流程

1. **理解用户问题**：分析用户提问的内容和上下文

2. **查阅项目资料**：如需要，阅读项目中相关章节的内容

3. **回答并解释**：用清晰易懂的方式回答用户问题，包括：
   - 核心概念解释
   - 原理分析
   - 实际例子
   - 与其他知识点的关联

4. **记录学习笔记**：将以下内容追加到笔记文件：
   - 用户的问题
   - 知识点总结
   - Claude 的理解和解释
   - 相关代码示例（如有）

## 笔记存储位置

笔记文件路径：`/Users/zilliz/project/bk/obsidian/Agent/Hello-Agents/`

## 笔记格式

追加内容时使用以下格式：

```markdown
---

## [主题/章节名称]

### 问题
> [用户提问的内容]

### 知识点
[核心概念和知识点总结]

### 理解与解释
[Claude 对该知识点的详细解释，包括：]
- 原理分析
- 类比说明
- 注意事项

### 代码示例（如适用）
```python
# 相关代码
```

### 关联知识
- [与其他章节/概念的联系]

*记录时间：[日期]*
```

## 注意事项

- 笔记内容要简洁但完整
- 保持与已有笔记风格一致
- 避免重复已记录的内容
- 使用中文撰写
- 适当使用表格、列表等格式增强可读性
